# Recipe Android App

A simple Android application for browsing and managing recipes.

## Features

- Browse recipe list
- View recipe details
- Search recipes
- Add to favorites
- Material Design UI

## Tech Stack

- Kotlin
- Android SDK (API 21+)
- Material Design Components
- RecyclerView for lists

## Getting Started

1. Open in Android Studio
2. Sync Gradle
3. Run on emulator or device
